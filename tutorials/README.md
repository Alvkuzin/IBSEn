This is the folder containing the examples of the project usage.

The examples are organized as jupyter notebooks. There are also some .jpg pictures for illustrations
 in the examples, and one movie `test_lc.mp4` which is produced near the end of `Full_tutorial`.

If you are new to the project, the best idea is to start with `Full_tutorial`. There, I collected the overview of almost everything the code can do.

Then, there are more detailed explanations about:
 1. Absorption, including photoelectric and gamma-gamma (not included in the `Full_tutorial` at all);
 2. Orbit of a binary;
 3. Winds (meaning outflows of the optical star and the pulsar);
 4. IBS
 5. Evolution of electrons in a stationary case for an extended emission zone;
 6. Evolution of electrons in a non-stationary case for a single emission zone;
 7. Non-thermal spectrum calculation.
